Introduction

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML.

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

  • CSS stands for Cascading Style Sheets
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
Getting Started

To start using CSS, you need to create a CSS file and link it to your HTML file.

Here is an example of how to link a CSS file to an HTML file:

link rel="stylesheet" href="styles.css"
  • Create a CSS file with a .css extension
  • Link the CSS file in the head section of your HTML file using a link element.
Basic Concepts

CSS consists of selectors and declarations. Selectors are used to select the elements you want to style.

Declarations are used to specify the styles for the selected elements. A declaration consists of a property and a value.

selector { property: value; }
  • Selectors: Used to select HTML elements.
  • Properties: Used to define what aspect of the element you want to style (e.g., color, font-size).
  • Values: Used to specify the value of the property (e.g., red, 16px).
Advanced Topics

Advanced CSS topics include animations, transitions, and responsive design.

Using media queries, you can create a responsive design that adapts to different screen sizes.

@media screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
  • Animations: Create complex animations using @keyframes.
  • Transitions: Smoothly transition from one style to another.
  • Media Queries: Apply different styles for different devices or screen sizes.
API Reference

Here are some commonly used CSS properties:

color: Sets the color of the text.

p { color: blue; }

background-color: Sets the background color of an element.

div { background-color: yellow; }
  • font-size: Sets the size of the font.
  • margin: Sets the margin outside an element.
  • padding: Sets the padding inside an element.
